# Changelog

Semua perubahan penting pada proyek **Invezgo MCP** akan didokumentasikan di file ini.

## [1.0.0] - 2026-06-07

### Added

- Peluncuran perdana (Initial Release) server MCP lokal Invezgo.
- **Stock Tools**: Registrasi 50 tools untuk penarikan data pasar saham Indonesia (profil emiten, grafik harga, indikator teknikal bandarmologi, komposisi pemegang saham, transaksi bursa harian, order book, kalender aksi korporasi, rotasi sektor, dll).
- **Personal Tools**: Registrasi 9 tools untuk mengakses data personal pengguna platform Invezgo (watchlist, portfolio, journal, trade log, trade summary chart, dll) secara read-only menggunakan otorisasi API Key.
- Penambahan file `LICENSE` (ISC License).
- Pembuatan file `README.md` lengkap beserta kebijakan privasi detail dan contoh penggunaan.
- Konfigurasi manifest versi `0.3` lengkap dengan validasi ikon dan metadata.
